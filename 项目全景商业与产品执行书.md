# 项目代号：【太和】(Taihe)

**—— AGI 时代的空间化多智能体调度操作系统**

## 一、 项目本质（一句话电梯演讲）

**“把看不见的 AI 黑盒，变成看得见的数字沙盘。”** 【太和】是一个基于“中式现实管理哲学”打造的多智能体（Multi-Agent）可视化操作系统。它通过将复杂的 AI 工作流降维映射为 **2D 像素风的现实办公空间**，让普通管理者能够像玩沙盘游戏一样，直观地调度、监控和干预 AI 团队，彻底解决企业在 AGI 时代的“技术黑盒焦虑”与“失控恐慌”。

## 二、 核心痛点与解决方案

### 1. 行业痛点：Agent 时代的“黑盒恐慌”

- **不可视**：当前的 AI 都在后台默默运行，老板不知道它在思考什么，更不知道它为什么卡住。
    
- **不敢用**：赋予 AI 执行权限（如发邮件、写代码、动用资金）极度危险，一旦代码死循环或产生幻觉，将造成严重损失。
    
- **门槛高**：现有的 Agent 编排工具（如 Coze, Dify）依然是面向程序员的“连线节点图”，普通管理者完全看不懂。
    

### 2. 【太和】的降维打击（解决方案）

- **交互降维（GUI for Agent）**：就像 Windows 用“桌面和回收站”替代了 DOS 命令行，我们用**“中式办公房间、红木桌、走动的小人”**替代了复杂的系统日志与代码流。
    
- **情绪对冲（心理安全感）**：将冰冷的报错，转化为拟人化的“小人被关进天牢等待御批”。用极具仪式感和趣味性的方式，化解管理者的技术焦虑。
    

## 三、 核心产品架构 (双核驱动)

项目采用“前端视觉感知 + 后端制度引擎”的双核架构：

### 1. 视觉感知层（前端：文明的窗口）

- **美术风格**：中国现代写实办公环境的 32 位像素风（拒绝赛博朋克）。红木办公桌、功夫茶具、实体机柜、带中文标识的警戒线。
    
- **技术栈**：`React 18` (负责侧边栏统帅控制台、数据面板) + `Phaser 3` (负责 2D 游戏沙盘渲染、小人寻路动画)。
    
- **空间划分（五部制）**：
    
    1. **统帅台（中枢）**：接收自然语言指令，分发任务（丞相 Agent 驻地）。
        
    2. **朝会厅（事件总线）**：Agent 之间互相交换数据和协作的区域。
        
    3. **藏书阁（RAG/数据源）**：调取外部搜索引擎、内部知识库的实体化区域。
        
    4. **开发区（执行引擎）**：编写代码、执行具体业务逻辑的区域。
        
    5. **隔离区/天牢（风控沙盒）**：存放报错、越权或陷入死循环的 Agent。
        

### 2. 制度引擎层（后端：生产关系的映射）

- **核心逻辑**：基于 `LangGraph` 或自研状态机（State Machine）搭建。
    
- **动作映射协议**：后端 LLM 每思考一步，都会向前端下发一条 JSON 协议，将逻辑节点转化为物理坐标的移动（如：后端检索数据库 = 前端小人走向藏书阁）。
    
- **Human-in-the-Loop（人机干预）**：当“刑部（风控 Agent）”拦截到敏感操作时，工作流挂起，前端小人瞬移至“天牢”，直到统帅（用户）在 React 面板点击“驳回”或“特赦”。
    

## 四、 商业变现模式

【太和】卖的不是聊天机器人，而是**“数字生产关系”与“组织管理能力”**。

1. **数字员工席位费 (MaaS - Management as a Service)**
    
    - 基础版（免费）：提供基础的办公空间和单体 Agent。
        
    - 企业版（SaaS订阅）：支持多 Agent 协同，按“同时在线的小人数量”及“空间大小”收费。
        
2. **“帝国模板”资产交易 (Agent App Store)**
    
    - 用户可以配置自己的“电商协作模式”、“自媒体流水线模式”。
        
    - 这些组织架构可以被打包成“装修模板”在平台售卖，平台抽成。用户一键购买，即可拥有一套成熟的数字团队。
        
3. **私有化风控底座 (Sovereign AI Infrastructure)**
    
    - 面向金融、政务等高合规要求企业。卖点是**“100% 过程可视化与审计”**。任何一笔数据流转，都有“史官 Agent”记录，并在像素沙盘中可追溯。
        

## 五、 项目演进路线图 (Roadmap)

### 第一阶段：MVP 最小可行性产品 (近 3 个月) —— “一纸风行”

- **目标**：跑通核心视觉工作流与风控拦截，用于投资人路演。
    
- **交付**：
    
    - 静态的中式实景 2D 像素沙盘。
        
    - 输入一句话，展示 1-2 个小人的接旨、走动、执行动作。
        
    - **高光时刻**：展示小人触发风控，被关入“天牢”，并由用户手动点击“释放/驳回”的完整交互。
        

### 第二阶段：可生长的数字组织 (1 年内) —— “百官归位”

- **目标**：完善 Agent 的编排能力，支持用户自定义“部门”和工作流。
    
- **交付**：
    
    - 引入拖拽式的空间配置：摆放一张“红木桌”等于实例化一个 Agent。
        
    - 上线“藏书阁”记忆系统，沉淀企业的 SOP。
        

### 第三阶段：全真互联的万物中枢 (未来) —— “数字第二平面”

- **目标**：接入 Computer Use 和物理世界 IoT。
    
- **交付**：统帅在系统内下达的指令，直接驱动现实世界的电脑操作、工厂机器和金融交易。各用户的“紫禁城”之间可以互相联通，形成分布式的数字文明。
    

## 六、 投资亮点总结

1. **开创了 AGI 时代的新交互标准**：用空间化、可视化的方式，解决了 LLM 落地的最后一公里痛点。
    
2. **极高的用户留存与心理黏性**：电子宠物/挂机游戏的陪伴感 + 真实的生产力输出。
    
3. **构建生态的潜力**：通过定义“部门”和“协作规则”，掌握了 AI 时代的数字生产关系标准。
